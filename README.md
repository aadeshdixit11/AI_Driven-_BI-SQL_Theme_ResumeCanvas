# AI_Driven-_BI-SQL_Theme_ResumeCanvas

## Overview
This project showcases an **AI-powered résumé** designed with a **SQL theme** and **KPI dashboard-style visualization**.  
Instead of a traditional text résumé, it leverages multiple AI tools to generate, refine, and present career data in an interactive and analytical format.

---

## Key Features
- **AI Collaboration**: Built entirely through the integration of Copilot, Gemini, GPT, Notebook LM, and Genspark AI.  
- **SQL Theme Integration**: Resume structure inspired by database schema and query logic.  
- **KPI Dashboard Visualization**: Highlights career metrics such as years of experience, projects completed, certifications, and skills.  
- **Multi-Format Output**: Available in HTML, PNG, and PDF formats for easy sharing.  
- **Case Study Included**: Documentation of the architecture and workflow behind the project.  

---

## Tools & Technologies
- **Copilot** → Prompt creation for SQL-based résumé theme.  
- **Gemini** → Generated canvas and final HTML code.  
- **GPT** → Refined code into dashboard-style visualization.  
- **Notebook LM** → Tested and validated execution.  
- **Genspark AI** → Converted HTML résumé into PDF format.
